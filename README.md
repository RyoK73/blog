# DEV活

RyoK73の趣味・開発活動・開発記録を記すパーソナルブログ

## ブログ記事コマンド

### 記事作成

```bash
pnpm post
```

### 記事編集

```bash
pnpm post --edit
```

## makrdown記法

| 記法           | 意味             |
| -------------- | ---------------- |
| `# ～ ######`  | 見出し H1〜H6    |
| `**text**`     | 太字             |
| `*text*`       | 斜体             |
| `~~text~~`     | 取り消し線       |
| `` `code` ``   | インラインコード |
| `> text`       | 引用             |
| `- / * / +`    | 箇条書き         |
| `1.`           | 番号付きリスト   |
| `- [ ]`        | チェックボックス |
| `[text](url)`  | リンク           |
| `![alt](url)`  | 画像             |
| `---`          | 水平線           |
| `\|col\|col\|` | テーブル         |
